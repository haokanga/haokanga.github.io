We alway meet PRESS HOLD RELEASE "FIRE" button to control our weapon
{% highlight C# %}OnPressAction(){% endhighlight %}triggers when you PRESS "FIRE" button
{% highlight C# %}OnHoldAction(){% endhighlight %}triggers when you HOLD "FIRE" button
{% highlight C# %}OnReleaseAction(){% endhighlight %}triggers when you RELEASE "FIRE" button

wanna monitor other keys? Here is the trick.
If the player press UP button (only the activity of PRESS, not HOLD or RELEASE) {% highlight C# %} if (Duck.inputProfile.Pressed("UP", false)) { ... } {% endhighlight %}
the button can be {% highlight C# %} "UP", "RIGHT", "DOWN", "LEFT", "JUMP", "LTRIGGER", "RAGDOLL", "STRAFE" "SHOOT" // not "fire" "GRAB" "QUACK" {% endhighlight %}
Updated version of PRESS HOLD RELEASE button table
{% highlight C# %}duck.inputProfile.Pressed("DOWN", false){% endhighlight %}triggers when you PRESS button
{% highlight C# %}this._equippedDuck.inputProfile.Down("LEFT"){% endhighlight %}triggers when you HOLD button {% highlight C# %} this._equippedDuck {% endhighlight %} get the owner(Duck) of this equipment
{% highlight C# %} if (this.inputProfile.Released("QUACK")) { this.quack = 0; } {% endhighlight %}triggers when you RELEASE button
Notice the object must be {% highlight C# %}Duck{% endhighlight %}.
and the result is a bool value.
